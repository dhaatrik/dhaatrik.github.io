# Technology Stack

## Core

- **Framework**: Astro (Static Site Generation)
- **Language**: TypeScript / JavaScript
- **Package Manager**: npm

## Styling & UI

- **CSS Framework**: Tailwind CSS v4
- **Styling Strategy**: Vanilla CSS Variables, Bento Grid
- **Typography**: Poppins (Headings) & Nunito (Body)

## Content Management

- **Format**: Markdown / MDX Collections

## Infrastructure

- **Deployment**: Automatic CI/CD to GitHub Pages
