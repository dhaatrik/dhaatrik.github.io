# Dhaatrik.

Hey! I’m Dhaatrik, a 26-year-old entrepreneur and developer from India, building the next generation of tech-driven solutions. I’m a first-principles thinker who believes that the best products come from understanding the absolute fundamentals.

This repository (`dhaatrik.github.io`) houses my personal portfolio and blog, built with **Astro** for extreme speed and **Tailwind CSS v4** for a sleek, glassmorphic dark-mode aesthetic.

👉 **View it live**: [https://dhaatrik.github.io](https://dhaatrik.github.io)

## 🚀 Tech Stack Highlights

- **Framework**: Astro (Static Site Generation)
- **Styling**: Tailwind CSS v4 + Vanilla CSS Variables (Bento Grid)
- **Typography**: Poppins (Headings) & Nunito (Body)
- **Content**: Markdown/MDX Collections
- **Deployment**: Automatic CI/CD to GitHub Pages

## 📂 Project Structure

- `src/pages/` - Core Astro routing (Home, About, Blog).
- `src/components/` - Reusable UI elements (Header, Footer, Bento Cards).
- `src/content/projects/` - The MD files generating the "Pretty-Good Stuffs" section.
- `src/styles/` - Global styling, CSS themes, and animation keyframes.

## 🧞 Development Setup

If you want to run this portfolio locally:

```bas
npm install
npm run dev
```

To build for production:

```bas
npm run build
```
